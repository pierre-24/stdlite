    stdl_wavefunction_compute_density_dsp¤

    int stdl_wavefunction_compute_density_dsp(size_t nocc, size_t nmo, size_t nao, double *C, double *D)

    Compute the density matrix (sy format).

    \[P_{\mu\nu} = \sum_r^{MO} n_r\,C_{r\mu}\,C_{r\nu},\]

    where \(n_r\) is the occupation number of MO \(r\).

    Parameter C
    double[nmo*nao], the coefficients
    Parameter nocc
    number of occupied orbitals, must be >0.
    Parameter nmo
    number of MO, must be >0.
    Parameter nao
    number of AO, must be >0.
    Parameter D
    double[STDL_MATRIX_SP_SIZE(nao)] the density matrix.
    Return
    error code.

    stdl_wavefunction_new¤

    int stdl_wavefunction_new(size_t natm, size_t nocc, size_t nao, size_t nmo, stdl_wavefunction **wf_ptr)

    Create a new wavefunction.

    Parameter natm
    number of atom (should be >0)
    Parameter nocc
    number of occupied orbitals, must be >0.
    Parameter nao
    number of atomic orbitals (should be >0)
    Parameter nmo
    number of molecular orbitals (should fulfill nelec <= 2*nmo <= 2*nao)
    Parameter wf_ptr
    wavefunction object to be initialized
    Return
    SDTL_ERR_OK if everything was ok.

    stdl_wavefunction_orthogonalize_C_dge¤

    int stdl_wavefunction_orthogonalize_C_dge(size_t nmo, size_t nao, double *S, double *C)

    Symmetrize the LCAO coefficients in place using a Löwdin orthogonalization.

    In other words:

    \[C^{\perp} = S^{1/2}\,C.\]

    See, e.g., there.

    Parameter nmo
    number of MO, must be >0.
    Parameter nao
    number of AO, must be >0.
    Parameter S
    double[STLD_MATRIX_SP_SIZE(nao)], the overlap matrix
    Parameter C
    double[nmo*nao], the coefficients to be orthogonalized
    Return
    the error code
