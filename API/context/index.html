    stdl_context_new¤

    int stdl_context_new(stdl_wavefunction *wf, stdl_basis *bs, float gammaJ, float gammaK, float ethr, float e2thr, float ax, stdl_context **ctx_ptr)

    Create a new context. Based on the information provided as input, a subset of MO from the wavefunction is selected, following the selection rules of sTD-DFT.

    Parameter wf
    A valid wavefunction.
    Parameter gammaJ
    parameter for Coulomb integrals approximation
    Parameter gammaK
    parameter for exchange integrals approximation
    Parameter ethr
    energy threshold for primary CSFs, must be >0
    Parameter e2thr
    energy threshold for secondary CSFs (selected pertubatively), must be >0
    Parameter ax
    amount of HF exchange
    Parameter ctx_ptr
    The context to be created.
    Return
    error code

    stdl_context_select_csfs_monopole¤

    int stdl_context_select_csfs_monopole(stdl_context *ctx, int compute_B)

    Select CSFs and build the \(\mathbf A\) and \(\mathbf B\) matrices, using the monopole approximation (original sTD-DFT). If B is set to NULL, then only \(\mathbf A\) is filled (Tamm-Dancoff approximation).

    Parameter ctx
    a valid context
    Parameter nselected
    number of CSFs that were selected. If equals to 0, then csfs and A are not initialized.
    Parameter csfs
    size_t[nselected], the indices (i*ctx->nvirt + a) of each selected CSF i→a, as i = csfs[k] / ctx->nvirt; a = csfs[k] % ctx->nvirt. They are sorted in increasing energy order, the energy being available at A[k * nselected + k].
    Parameter A
    float[STDL_MATRIX_SP_SIZE(nslected)], part of the electronic Hessian matrix, in sp format, to be created. Caller is responsible for free'ing it.
    Parameter B
    float[STDL_MATRIX_SP_SIZE(nslected)], part of the electronic Hessian matrix, in sp format, to be created. Might be NULL if only A is required. If not, caller is responsible for free'ing it.
    Return
    error code

    stdl_context_select_csfs_monopole_direct¤

    int stdl_context_select_csfs_monopole_direct(stdl_context *ctx, int compute_B)

    Select CSFs and build the \(\mathbf A\) and \(\mathbf B\) matrices, using the monopole approximation (original sTD-DFT). If B is set to NULL, then only \(\mathbf A\) is filled (Tamm-Dancoff approximation). Use less memory but takes more time than the non-direct version.

    Parameter ctx
    a valid context
    Parameter nselected
    number of CSFs that were selected. If equals to 0, then csfs and A are not initialized.
    Parameter csfs
    size_t[nselected], the indices (i*ctx->nvirt + a) of each selected CSF i→a, as i = csfs[k] / ctx->nvirt; a = csfs[k] % ctx->nvirt. They are sorted in increasing energy order, the energy being available at A[k * nselected + k].
    Parameter A
    float[STDL_MATRIX_SP_SIZE(nslected)], part of the electronic Hessian matrix, in sp format, to be created. Caller is responsible for free'ing it.
    Parameter B
    float[STDL_MATRIX_SP_SIZE(nslected)], part of the electronic Hessian matrix, in sp format, to be created. Might be NULL if only A is required. If not, caller is responsible for free'ing it.
    Return
    error code
